# Камера хранения (RGZ)

Сайт: камера хранения с бронированием ячеек.

## Функции
- Неавторизованный: видит свободно/занято и статистику, но данные о пользователях скрыты.
- Авторизованный: может бронировать свободные ячейки (до 5), отменять свои брони, видеть кто занял ячейку.
- Администратор: может очистить все брони и просмотреть список пользователей.

## Технологии
Backend: Flask + PostgreSQL + JWT  
Frontend: HTML + CSS + JavaScript (перезагрузка страниц после действий)
